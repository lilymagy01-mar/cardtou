# 📸 고객 사진 카드 커버 제작 & 출력 기능 개발 계획서
**작성일:** 2026-04-10 | **총괄:** Captain Flora (캡틴 플로라)

---

## 1. 프로젝트 목표

고객이 제공한 사진을 **버튼 하나로** 자동 보정·합성하여, 인쇄 품질의 카드 커버를 즉시 생성·출력하는 전용 워크플로우 구축.

> **핵심 철학:** 정밀 슬라이더 없이, 모든 편집은 **1버튼 = 1효과** 방식으로 자연스러운 최적값 자동 적용.

---

## 2. 에이전트 역할 분담표

| 에이전트 | 담당 영역 | 이번 기능에서의 역할 |
|---|---|---|
| 🎭 Captain Flora | 총괄 디렉터 | 워크플로우 설계, QC, 단계별 승인 |
| 🎨 Arty | 비주얼 프롬프트 | 사진 분위기 분석 → AI 배경/장식 프롬프트 생성 |
| 🖼️ Template Curator (샘플리) | 이미지 합성 | 누끼 레이어 + 프레임 + 배경 합성 처리 |
| 📐 Layout Architect (각도기) | 규격 설계 | 사진 위치/크기 좌표 산출, Safe Zone 보장 |
| 🔤 Typography Wizard (폰트김) | 텍스트 렌더링 | 카드 위 메시지 자동 배치 및 폰트 최적화 |
| 🖨️ Print Commander (프린트박) | 출력 | 300DPI PDF 생성 및 프린터 전송 |

---

## 3. 전체 워크플로우 (4단계)

1단계: 사진 업로드
2단계: 자동 보정 & 원터치 편집 패널
3단계: 카드 커버 합성 (레이아웃 + 누끼 + 프레임)
4단계: PDF 생성 & 출력

---

## 4. 원터치 편집 도구 스펙 (핵심)

**원칙:** 각 버튼은 최적값이 내장되어 있어 클릭 즉시 자연스러운 결과 적용. 슬라이더·수치 입력 없음.

### 4-1. 자동 보정 도구 (Auto Enhance)

| 버튼명 | 효과 | 적용 알고리즘 |
|---|---|---|
| ✨ 자동보정 | 전체 밝기·대비·색상 한번에 최적화 | 히스토그램 균등화 + 채도 +15% |
| ☀️ 밝게 | 노출 자동 증가 | brightness +20, 하이라이트 보호 |
| 🎨 선명하게 | 윤곽 강조 | unsharp mask (radius:1, amount:0.8) |
| 🌙 따뜻하게 | 황금시간대 색온도 | color temp +300K |
| ❄️ 차갑게 | 블루톤 쿨한 느낌 | color temp -300K |
| 🌸 비네팅 | 가장자리 자연스럽게 어둠 처리 | radial gradient overlay opacity 40% |

### 4-2. 누끼 (Background Removal)

| 버튼명 | 효과 | 처리 방식 |
|---|---|---|
| ✂️ 누끼따기 | 인물/오브젝트 배경 자동 제거 | @imgly/background-removal (브라우저 WASM) |
| 🖼️ 누끼 + 배경 | 누끼 후 테마 배경 자동 합성 | 누끼 레이어 → 샘플리 배경 합성 |

### 4-3. 합성 & 프레임

| 버튼명 | 효과 | 처리 방식 |
|---|---|---|
| 🌸 꽃 프레임 | 꽃 장식 PNG 오버레이 | SVG/PNG 레이어 합성 (샘플리) |
| 💫 하트 프레임 | 하트 장식 오버레이 | 동일 |
| 🎀 리본 | 상단 리본 장식 | 동일 |
| 🖼️ 카드 테두리 | 엽서풍 테두리 적용 | CSS border or PNG mask |
| 🎭 AI 배경생성 | 아티가 사진 분위기 분석 → 배경 AI 생성 | Arty 프롬프트 → Imagen API |

### 4-4. 필터 (Instagram 스타일)

| 버튼명 | 프리셋 |
|---|---|
| 🌷 로맨틱 | 채도 +20, 핑크톤, 소프트 비네팅 |
| 🌿 내추럴 | 채도 -5, 그린톤, 선명도 +10 |
| ⬛ 블랙앤화이트 | 흑백 전환, 대비 +15 |
| 🌟 골든 | 황금색조, brightness +10, warmth +400K |
| 🎨 원본 | 모든 필터 제거 리셋 |

## 5. UI 레이아웃 설계

```
┌───────────────────────────────────────────────────┐
│  📷 고객 사진 업로드 & 편집                        │
│  ┌──────────────────┐  ┌──────────────────────┐   │
│  │                  │  │  자동보정  ┃  누끼    │   │
│  │  [사진 미리보기]  │  │────────────────────  │   │
│  │  (Canvas 렌더링) │  │  [필터 버튼 그리드]  │   │
│  │                  │  │────────────────────  │   │
│  │                  │  │  [프레임 버튼 그리드] │   │
│  └──────────────────┘  └──────────────────────┘   │
│                                                     │
│         [↩ 되돌리기]  [카드 커버에 적용 →]         │
└───────────────────────────────────────────────────┘
```

- 드래그 앤 드롭 또는 클릭 업로드 지원
- 업로드 즉시 자동보정 자동 실행 (opt-out 가능)
- Undo 1단계 지원 (직전 상태 즉시 복구)

## 6. 기술 스택

| 기능 | 라이브러리 / 방법 |
|---|---|
| 누끼 (배경 제거) | @imgly/background-removal (WASM, 서버 불필요) |
| 이미지 필터/보정 | HTML5 Canvas API + CSS filter 조합 |
| 이미지 합성 (프레임) | Canvas API drawImage 멀티 레이어 |
| AI 배경 생성 | Arty 프롬프트 → 기존 /api/ai/image 라우트 |
| PDF 생성 | 기존 Print Commander 파이프라인 |
| 사진 저장 | Supabase Storage (원본 + 편집본) |

## 8. 데이터 흐름

```
사용자 사진 업로드
  → PhotoUploader → Supabase Storage 임시 저장
  → autoEnhance() 자동 실행
  → PhotoEditPanel 표시
      → [누끼따기] → backgroundRemoval.ts (WASM)
      → [필터] → filterPresets.ts → Canvas.filter 적용
      → [프레임] → frameCompositor.ts → Canvas 레이어 합성
      → [AI배경] → Arty 프롬프트 → /api/ai/image 호출
  → [카드 커버에 적용] 클릭
      → 각도기: 카드 내 사진 배치 좌표 계산
      → EditorCanvas에 합성 이미지 삽입
      → 폰트김: 텍스트 레이어 자동 배치
  → [출력] 클릭
      → 프린트박: 300DPI PDF 생성 → 프린터 전송
```
